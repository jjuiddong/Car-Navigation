# Car-Navigation
Car Navigation on Windows10 Tablet PC with VS2017 C++, Direct3D11, ImGui

- Develop Toolkit
  - Visual Studio 2017, C++
- Library
  - Direct3D11
  - Assimp
  - ImGui
  - Poco Library
  - Boost
    - manual setup
  - MongoDB
  	- install by vcpkg
  		- ./vcpkg install mongo-cxx-driver
- Hardware
  - Lenovo D330
